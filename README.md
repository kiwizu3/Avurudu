A web app celebrating Sri Lanka's most beloved cultural festival — with live Nakath countdowns, ritual timings, and rich traditions.

🌐 Live Site → nakath.vercel.app

✨ About
Avurudu (අවුරුදු) is a React web application built to help Sri Lankans — and anyone who loves the culture — celebrate the Sinhala & Tamil New Year (April 13–14). The app provides accurate auspicious times (Nakath), ritual guides, and a live countdown to the New Year moment.
Whether you want to know the right time to light the hearth, boil milk, or exchange gifts, this app has you covered.

🪔 Features

Live Countdown — Counts down to the precise New Year moment
Nakath Timings — Full list of auspicious times for each ritual
Cultural Traditions — Information about Avurudu customs and practices
Multilingual Support — Content in Sinhala and Tamil
Responsive Design — Works beautifully on mobile and desktop
Smooth Animations — Powered by Framer Motion for a polished feel
SEO Optimised — Open Graph, Twitter Card, and structured data included

🚀 Getting Started
Prerequisites

Node.js v18+
npm or yarn

Installation
bash# Clone the repository
git clone https://github.com/kiwizu3/Avurudu.git
cd Avurudu

# Install dependencies
npm install

# Start the development server
npm run dev
The app will be available at http://localhost:5173.
Build for Production
bashnpm run build
Output is generated in the dist/ folder, ready to deploy to any static hosting provider.
Preview Production Build
bashnpm run preview
